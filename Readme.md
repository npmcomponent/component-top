# top

  Transient back-to-top link that fades in once
  the user has scrolled more than 50% of the page
  height.

  ![back to top link](http://f.cl.ly/items/4101392h0C2Y1p1x0O25/top.png)

## Installation

    $ component install component/top

## Example

```js
require('top')();
```
